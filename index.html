دستورات ضروری گیت


در این صفحه دستورات کاربردی و ضروری گیت قرار داده می شود

دستورات پایه

اضافه کردن گیت به یک پروژه:

git init

مشاهده وضعیت:

git status

اضافه کردن:

git add "FileName"

اضافه کردن همه تغییرات به صورت یکجا:

git add .

کامیت کردن:

git commit -m "CommitNote"


لاگ گرفتن

دستور اصلی

git log

لاگ با اطلاعات بیشتر:

git log --stat
note: press q key for exit.

نمایش لاگ ها در یک خط:

git log --oneline


کلون کردن

انتقال یک پروژه از منبع خارجی به لوکال به صورت کامل ، در اینجا مثال گیت هاب می باشد:

git clone "addres project in github" "new name project in my system"


Branch

نمایش همه Branch ها:

git branch

یا

git branch --list

حذف یک Branch:

git branch -d "BranchName"


Push , Remote و pull

نمایش همه ریموت ها:

git remote

اضافه کردن یک ریموت جدید:

git remote add "RemoteName" "address remote for example github"

پوش کردن:

git push "remote name" "branch name in remote"

حدف یک ریموت:

git branch -d "BranchName"
